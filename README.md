***Daila*** is a command line application for quickly and easily recording and visualizing daily data. 

## Installation

### Cargo
```bash
$ cargo install daila
```

### From source
```bash
$ git clone https://github.com/DevinLeamy/daila
$ cd daila
$ cargo build [--release]
```
Then add '***$PWD/target/debug/***' or '***$PWD/target/release/***' to your path.

## Usage
```bash
$ daila
```
